# [iOS/SwiftUI] 화면 레이아웃의 핵심: HStack, VStack, ZStack

## 1. SwiftUI 레이아웃의 기본 철학
* SwiftUI는 절대 좌표(X, Y)를 직접 계산해서 화면을 그리는 방식이 아니다.
* 큰 상자(Stack) 안에 UI 요소(View)들을 **어떤 방향으로 차곡차곡 쌓을 것인가**를 결정하는 상대적 배치 방식을 사용한다.
* 이는 컴퓨터 공학의 **트리(Tree) 자료구조**와 동일한 원리다.

---

## 2. 3가지 핵심 스택(Stack)

### ① HStack (Horizontal Stack) : 수평 배치
* **개념:** 내용물들을 왼쪽에서 오른쪽으로, 가로 방향으로 나란히 줄 세운다.
* **실전 활용:** 아이콘 바로 옆에 과목 이름(예: '컴퓨터 구조')을 나란히 적을 때 사용한다.

### ② VStack (Vertical Stack) : 수직 배치
* **개념:** 내용물들을 위에서 아래로 차곡차곡 쌓아 올린다.
* **실전 활용:** 과목 이름 바로 아래에 '오늘의 누적 학습 시간'을 두 줄로 배치할 때 사용한다.

### ③ ZStack (Depth Stack) : 겹쳐서 배치
* **개념:** 3차원 Z축(깊이)을 기준으로, 도화지 바닥부터 사용자 눈앞으로 겹겹이 포개어 올린다. 나중에 작성한 코드가 가장 위(최상단)에 올라온다.
* **실전 활용:** 프로필 사진 위에 상태 배지를 얹거나, 원형 타이머 그래프 정중앙에 달성률(%) 텍스트를 겹쳐서 띄울 때 사용한다.
