## 📌 점근 표기법 정의

| 표기 | 의미 | 조건 |
|--|--|--|
| f(n) ∈ O(g(n)) | g(n)이 **상한** | ∃c, n₀ > 0 s.t. f(n) ≤ c·g(n) for all n ≥ n₀ |
| f(n) ∈ Ω(g(n)) | g(n)이 **하한** | ∃c, n₀ > 0 s.t. f(n) ≥ c·g(n) for all n ≥ n₀ |
| f(n) ∈ Θ(g(n)) | **상한 + 하한** | O(g(n)) 이고 Ω(g(n)) |
| f(n) ∈ o(g(n)) | **엄격한 상한** | lim f(n)/g(n) = 0 as n → ∞ |

> **핵심**: c와 n₀는 양의 상수이기만 하면 어떤 값이든 OK. 존재만 하면 증명 성립!

## 1. 최댓값 알고리즘

### 아이디어
- `max` 변수를 첫 번째 원소로 초기화
- 두 번째 원소부터 순회하며 `max`보다 크면 업데이트
- 루프 종료 시 `max` = 전체 리스트의 최댓값 보장

### 코드 (C++)

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    int max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }

    cout << max << endl;
    return 0;
}
```

### 시간복잡도
- **O(n)**: 리스트를 한 번 순회하며 n-1번 비교
- **최적(optimal)**: 최댓값을 찾으려면 모든 원소를 반드시 한 번씩 확인해야 하므로, 어떤 알고리즘도 O(n)보다 빠를 수 없음

## 2. f(n) = n² + 3n³ ∈ Θ(n³) 증명

### O(n³) 증명

n² + 3n³ ≤ c·n³ 을 만족하는 c, n₀를 찾는다.

양변을 n³으로 나누면:

$$\frac{1}{n} + 3 \leq c$$

c = 4로 잡으면 1/n ≤ 1 이므로 **n ≥ 1일 때 성립**.

> c = 4, n₀ = 1 → n ≥ 1인 모든 n에 대해 n² + 3n³ ≤ 4n³ ✓

### Ω(n³) 증명

n² + 3n³ ≥ c·n³ 을 만족하는 c, n₀를 찾는다.

양변을 n³으로 나누면:

$$\frac{1}{n} + 3 \geq c$$

c = 3으로 잡으면 1/n + 3 ≥ 3, 즉 1/n ≥ 0 이므로 **모든 n ≥ 1에서 성립**.

> c = 3, n₀ = 1 → n ≥ 1인 모든 n에 대해 n² + 3n³ ≥ 3n³ ✓

### 결론
> f(n) ∈ O(n³) 이고 f(n) ∈ Ω(n³) 이므로 **f(n) ∈ Θ(n³)** ✓

## 3. 점근 표기식 판별

### (a) lg n ∈ O(n) — **참** ✓

n은 lg n보다 훨씬 빠르게 증가한다.

c = 1, n₀ = 1로 잡으면 n ≥ 1인 모든 n에 대해 lg n ≤ n 성립.

### (b) n ∈ O(n lg n) — **참** ✓

n ≤ c·(n lg n) 양변을 n으로 나누면 1 ≤ c·lg n.

c = 1, n₀ = 2로 잡으면 lg 2 = 1 이므로 성립.

### (d) 2ⁿ ∈ Ω(5^(lg n)) — **참** ✓

밑 변환 공식으로 단순화:

$$5^{\log_2 n} = n^{\log_2 5} \approx n^{2.32}$$

지수함수 2ⁿ은 다항함수 n^2.32보다 훨씬 빠르게 증가한다.

c = 1/4, n₀ = 1로 잡으면 n ≥ 1인 모든 n에 대해 2ⁿ ≥ (1/4)·n^2.32 성립.

### (e) lg n³ ∈ o(n^0.5) — **참** ✓

소문자 o 조건: lim f(n)/g(n) → 0 임을 보인다.

lg n³ = 3 lg n 으로 단순화 후 극한 계산:

$$\lim_{n \to \infty} \frac{3\log_2 n}{n^{0.5}}$$

∞/∞ 꼴이므로 **로피탈 법칙** 적용:

$$= \lim_{n \to \infty} \frac{3/(n \cdot \ln 2)}{1/(2\sqrt{n})} = \lim_{n \to \infty} \frac{6\sqrt{n}}{n \cdot \ln 2} = \lim_{n \to \infty} \frac{6}{\sqrt{n} \cdot \ln 2} = 0$$

∴ lg n³ ∈ o(n^0.5) ✓

## 📎 자주 쓰는 미분 공식

| 함수 | 미분 |
|--|--|
| c (상수) | 0 |
| xⁿ | n·xⁿ⁻¹ |
| eˣ | eˣ |
| ln x | 1/x |
| log_a x | 1/(x · ln a) |
| log_a xᵏ | k/(x · ln a) |

### Chain Rule (합성함수)
> [f(g(x))]' = f'(g(x)) · g'(x)

예: (ln x²)' = (1/x²) · 2x = 2/x

### Product Rule (곱의 미분)
> [f·g]' = f'g + fg'

예: (x² · ln x)' = 2x·ln x + x²·(1/x) = x(2ln x + 1)

### Quotient Rule (몫의 미분)
> [f/g]' = (f'g - fg') / g²

예: (x / ln x)' = (ln x - 1) / (ln x)²
