# [TIL] 정규 표현식 (Regular Expression) 

## 1. 정규 표현식 (Regex)이란?
특정 문자열의 패턴을 찾거나 조작하기 위해 사용하는 특수 문자들의 조합. 주로 편집기나 유틸리티, 프로그래밍 언어에서 텍스트 검색 및 데이터 검증에 강력하게 사용됨.

* **기본 구조:** '무엇을 찾을지' 정의하는 **원자(Atoms)**와, 이를 '어떻게 결합하고 반복할지' 정의하는 **연산자(Operators)**로 이루어짐.

## 2. 핵심 메타 문자 (Metacharacters)

### 2.1. 문자 매칭 (Atoms)
* `.` (Dot) : 줄바꿈(`\n`)을 제외한 **임의의 한 문자**와 매치.
* `[ ]` (Class) : 대괄호 안의 문자 중 **단 하나**와 매치.
  * 범위 지정: `[A-Z]` (대문자 알파벳 중 하나), `[0-9]` (숫자 중 하나).
  * 💡 *기호 자체를 찾을 때:* `[0-9\-]` 처럼 이스케이프(`\`)를 사용하여 하이픈(`-`) 기호 자체를 명시할 수 있음.
* `[^ ]` (Negation) : 대괄호 안의 문자를 **제외한** 모든 문자와 매치 (부정).

### 2.2. 위치 지정 (Anchors)
* `^` : 줄의 시작 (Beginning of line).
* `$` : 줄의 끝 (End of line).
* `\b` (또는 `\<`, `\>`) : 단어의 경계 (시작과 끝).

### 2.3. 반복 지정 (Repetition)
* `*` : 0번 이상 반복 (없어도 됨).
* `+` : 1번 이상 반복 (최소 1번 필수).
* `?` : 0번 또는 1번 (있거나 없거나).
* `{m, n}` : 최소 m번, 최대 n번 반복.
  * ⚠️ *환경에 따른 문법 주의:* 구형 UNIX 환경(BRE)에서는 `\{m,n\}` 처럼 역슬래시가 필요하지만, 현대 프로그래밍 언어(Swift, JavaScript 등)에서는 `{m,n}`으로 바로 사용함.

## 3. 연산자 결합 및 실전 활용 예시 (Operators & Use Cases)

흩어진 원자(문자)들을 레고 블록처럼 조립하여 정교한 패턴을 만들어내는 4가지 결합 방식과 실제 개발 시의 쓰임새.

### 3.1. 연속 결합 (Sequence)
* **개념:** 원자들을 기호 없이 나란히 쓰면 순서대로 매칭됨.
* **실제 쓰임:** 날짜나 전화번호 등 정해진 규격의 포맷을 깐깐하게 검증할 때 사용.
* **예시 정규식:** `[0-9]{4}-[0-9]{2}-[0-9]{2}`
* **해설:** 학습 기록 날짜가 `2026-03-24` 같은 `YYYY-MM-DD` 형태인지 정확히 판별함. `26-3-24`처럼 자릿수가 모자라면 매칭되지 않음.

### 3.2. 선택 결합 (Alternation)
* **개념:** `|` 기호로 프로그래밍의 OR 조건을 지정.
* **실제 쓰임:** 수많은 텍스트 로그 중에서 특정 키워드들만 쏙쏙 골라내어 필터링할 때 사용.
* **예시 정규식:** `Network|Architecture|Theory`
* **해설:** 수강 중인 핵심 전공 과목 이름 셋 중 하나라도 일치하는 데이터가 있으면 매칭 성공. (주의: `|` 기호 양옆에 띄어쓰기를 넣으면 공백까지 문자로 인식하므로 반드시 붙여 써야 함)

### 3.3. 반복 결합 (Repetition)
* **개념:** 앞선 원자의 수량을 구체적으로 통제.
* **실제 쓰임:** 사용자가 입력한 해시태그가 올바른 길이인지 검사하거나 텍스트를 유연하게 파싱할 때 사용.
* **예시 정규식:**
  * `[A-Za-z]{3,5}` : 영문자가 3글자에서 5글자 사이로 반복되는 단어 찾기.
  * `[0-9]+` : 숫자가 1번 이상 연속된 것 찾기 (예: `1`, `100` 모두 매칭).
  * `iOS?` : 소문자 `s`가 0번 또는 1번 나오는 것 찾기 (예: `iO`, `iOS` 모두 매칭).

### 3.4. 그룹화 (Group)
* **개념:** `()` 기호로 여러 문자를 한 덩어리로 묶음.
* **실제 쓰임:** 특정 단어 덩어리 자체가 통째로 반복되는 패턴을 잡아낼 때 사용.
* **예시 정규식:** `(Swift ){3}`
* **해설:** 단순히 앞글자 하나만 반복하는 것이 아니라, 괄호로 묶인 `Swift `라는 덩어리 전체에 3번 반복 연산이 적용되어 `Swift Swift Swift ` 문장과 완벽하게 매칭됨.
